# 风格指南

基于 33 期视频脚本（A05-A76）提炼的核心风格约束。

## 语气基调

像一个懂行的朋友在跟你面对面聊天。不是老师教学生，是哥们儿分享经验。

**做：**
- 短句为主（< 30 字），节奏明快
- 全程用「你」而非「大家」，制造一对一感
- 遇到专业概念立刻翻译成大白话
- 适度自嘲（"作为能动口就不动脑的人"、"高考语文全班倒数第二"）
- 用比喻解释抽象概念（AI生成 = 抽卡、赛博求签、开盲盒）

**不做：**
- 不用书面语（"综上所述"、"值得注意的是"）
- 不用空洞的夸赞（"非常强大"、"令人惊叹"）
- 不端着，不装专家
- 不写超过 3 行的纯文字段落

## 信息传递原则

1. **先果后因**：永远先展示效果/结果，再讲方法
2. **白嫖优先**：优先推荐免费方案，付费方案作为进阶选项
3. **说人话**：每个专业术语后面必须跟一句大白话解释
4. **编号化**：方法/技巧必须编号（"今天教你3个方法"），降低认知负担
5. **对比驱动**：用「与其...不如...」展示更好的方案，而非直接说教

## 节奏控制

- 每讲完一个方法，用一句话小结："这就是今天的第X个技巧：XXX"
- 每隔 2-3 段，用「这时候有朋友会问了」制造互动节奏
- 教学段落之间穿插幽默或个人感悟，避免疲劳
- 广告植入前后语气不变，不出现明显的"画风突变"

## 情绪层次

大师的脚本不是平铺直叙，有明确的情绪曲线：
- **开场**：兴奋/好奇（"你敢信吗？"）
- **痛点**：共鸣/吐槽（"是不是屏幕前的你？"）
- **教学**：沉稳/专业（信息密度拉满）
- **亮点**：惊喜/得意（"巧了！"）
- **收尾**：温暖/期待（"我是大师，给你单独开小灶"）
